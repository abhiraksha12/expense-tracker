## 🚀 Features

* **Interactive Data Visualization:** Dynamic pie charts showing spending breakdowns by category and bar graphs tracking monthly trends.
* **Smart Transaction Logging:** Easily add, track, and review transactions with descriptive notes, accurate timestamps, and clear categorization.
* **Persistent Storage:** Fully integrated database system ensuring your financial data is safely stored and never lost when you close the app.
* **Export Capabilities:** Generates clean data sheets to review your complete transaction history externally.

---

## 🛠️ Tech Stack & Architecture

* **Frontend:** HTML5, CSS3 (Modern Dark Mode UI), JavaScript (Dynamic chart rendering and UI logic)
* **Backend:** Python (Application logic, request handling, and routing)
* **Database / Storage:** SQLite (`expenses.db`) for secure relational data and CSV integration (`expenses.csv`) for easy data sheets.
